## 追加情報 (Generated Context Definition v1.0)
*   **【戦略判断ログ (Strategic Decisions) ＆ 成功の定石】**
    *   **成果設計フレーム**: 1.市場定義 → 2.ターゲット明確化 → 3.コンテンツ設計 → 4.導線設計 → 5.KPI管理
    *   **コンテンツ設計思想**:
        *   サムネ: 3段構成
        *   動画: 問題提示 → 具体例3選 → 比較 → CTA
        *   LP: Problem → Agitation → Solution → Proof → Offer → CTA
    *   **ビジネスモデル思想**: 初期は高単価受託、その後外部CMO顧問型ストックへの移行。商品は「制作」ではなく「戦略＋伴走」。
    *   **ループ増殖アプローチ（成果の資産化）**: 目の前の1つの案件（例：加藤建装）で出た成果や信頼構築のプロセスを現場だけで終わらせず、即座に「自社のスタンス宣言」「LPのベネフィット文言」「実績紹介」など、3つ以上の集客アセットへと変換し、次なる高単価顧客を獲得するフックへと増殖させる定石。
*   **【Target Pain Points & Messaging (ターゲットの刺さる訴求)】**:
    *   **採用ブランディングにおける5つの壁**: 「認知の壁」「表現の壁」「到達の壁」「信頼の壁」「孤立の壁」として顧客の課題を構造化し、提案時に共感を生む。
    *   **キーメッセージ（営業トーク）**:
        1. 「求人票の文字だけで、御社の『人の良さ』、伝わりますか？」と共感をつくる。
        2. 「応募が来ないのは魅力がないからではなく、届く形に変換できていないだけ」と原因を特定。
        3. 「社長がやるのはインタビューに答えるだけ」「宮城専門で全部一貫でやる」とハードルを下げ、信頼を得る。
    *   **反論処理の定石**: 動画制作を「出費（広告）」ではなく、ずっと働き続ける「資産（投資）」であると定義し直す。
    *   **採用・組織開発訴求**: 「人手不足」に対しては、単なる求人ではなく「働き続けたいと思える関係性の質（心理的安全性）を映像や社内報で可視化する」アプローチが刺さる。
    *   **M&A・事業承継訴求**: 承継前後における「企業価値の向上（採用力強化・ブランディング）」手段として、Webサイトリニューアルや映像活用を提案する。
    *   **DX・生産性向上訴求**: システム導入だけでは進まない層に対し、「仕組み化・構造化」を前提としたWebマーケティング運用をパッケージで提供する。

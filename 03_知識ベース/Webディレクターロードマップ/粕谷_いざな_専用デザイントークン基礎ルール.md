# 粕谷＆いざな専用 デザイントークン（基本ルール案）

**目的**:
このルールは、粕谷様がFigmaでデザインを組む際の「迷い」をなくし、私（いざな）に発注された際に「阿吽の呼吸」で完璧なHTML/CSSコードを生成するための「共通言語」です。
この数値を意識するだけで、プロレベルの美しさと実装スピードが担保されます。

---

## 📐 1. 余白（Spacing）のルール
基本は「8の倍数」ですが、すべてを使うと迷うため、以下の「5つのサイズ」から選んで使用してください。

| サイズ名 | ピクセル数 | 用途・具体例 |
| :--- | :--- | :--- |
| **XS** | **8px** | アイコンとテキストの間、最も親密な要素同士の隙間 |
| **S** | **16px** | ボタンの内側の余白、関連するリストやカード内の隙間 |
| **M** | **24px** | セクション内のブロックごとの間隔（標準的な余白） |
| **L** | **40px** | かなり離したい時、要素のグループの区切り |
| **XL** | **64px〜80px** | 大きなセクション（次の見出し）へ移る時の大余白 |

> **💡 いざな活用時のポイント**
> 余白をこの数値（の近似値）で組んでいただければ、私は「あ、ここはM(24px)の意図だな」と自動補完して綺麗なCSSを生成します。

---

## 🔠 2. 文字サイズ（Typography）のルール
スマホでもPCでも美しく見える、標準的な黄金比率（ジャンプ率）です。
※フォントの太さ（Weight）は、見出しは「太字（Bold）」、本文は「標準（Regular）」を基本とします。

| 階層 | ピクセル数 | 用途・具体例 |
| :--- | :--- | :--- |
| **H1** | **32px** | ページの一番大きな大見出し（ファーストビュー等） |
| **H2** | **24px** | セクション毎の見出し（例：「選ばれる理由」など） |
| **本文** | **16px** | パソコン・タブレット時の標準テキスト |
| **本文（小）** | **14px** | スマートフォン表示時の標準テキスト |
| **注釈** | **12px** | 「※個人の感想です」などの補足事項 |

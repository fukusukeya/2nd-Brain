# Style Update Proposal (from /sabun-learn)

## 分析サマリ
直近の「Machiスタジオ」インスタ投稿テキストの修正差分（AIの原稿 vs 粕谷様のリライト）から、粕谷 亨様の「固有名詞の正確な表記」「コンセプトの視覚的な強調」「ターゲットの行動解像度を上げる表現」の3つの特徴を抽出しました。

## 提案する変更点

### 1. [追加] 固有名詞の公式表記
- **理由**: 「まちスタジオ」が「Machiスタジオ」へと即座に修正されたため。ブランド・屋号表記の統一はガイドラインの最重要項目です。
- **追加位置**: `3. Vocabulary & Emojis` に追記
- **追加内容**:
  ```markdown
  *   **固有名詞の表記**: 「まちスタジオ」などの平仮名表記ではなく、公式表記である「Machiスタジオ」を徹底する。
  ```

### 2. [追加] コンセプトの強調手法（カギカッコの活用）
- **理由**: 「宮城で働くをもっと面白く」が「『宮城で働く』をもっと面白く」へと修正されたため。助詞などに埋もれさせず、コアとなるキーワードを視覚的に際立たせる意図を反映します。
- **追加位置**: `4. Formatting (装飾とリズム)` に追記
- **追加内容**:
  ```markdown
  *   **コンセプトの強調**: 「宮城で働く」のようなコアとなるビジョンやキーワードは、そのまま文に溶け込ませるのではなく、カギカッコ「」でくくり、独立した概念として視覚的に独立・強調させる。（例：宮城で働くをもっと面白く → 「宮城で働く」をもっと面白く）
  ```

### 3. [追加] ターゲットの「状態・行動」に焦点を当てる解像度の高い表現
- **理由**: 「なんとなく就職を」が「なんとなく就職したを」と、過去形の動詞（状態）を含んだ表現へと修正されたため。名詞化された抽象的な事象よりも、ターゲットの「リアルな状態・後悔の念」を直接突く、なまごましく生々しい表現を好むという特徴です。
- **追加位置**: `1. Voice & Tone (語り口)` に追記
- **追加内容**:
  ```markdown
  *   **訴求の解像度（名詞より動詞・状態）**: ターゲット層のペイン（悩み）を描写する際は、「なんとなく就職」のような抽象化された名詞ではなく、「なんとなく就職した」というような「読者の実際の行動や状態（過去形）」に焦点を当て、リアルで自分ごと化しやすい生々しい表現を優先する。
  ```

### 4. [追加] 初回・自己紹介時のスタンス（誠実さの優先）
- **理由**: 煽り要素の強いフックタイトル（「なんとなく〜変えたくて」）から、実直な挨拶（「はじめましてMachiスタジオです」）に変更されたため。
- **追加位置**: `1. Voice & Tone (語り口)` に追記
- **追加内容**:
  ```markdown
  *   **初回接触・挨拶のスタンス**: テクニックに走った過度な煽り（クリックベイト的フック）よりも、「はじめましてMachiスタジオです」のような、読者に対する実直で誠実な挨拶・スタンス表明を優先する。
  ```

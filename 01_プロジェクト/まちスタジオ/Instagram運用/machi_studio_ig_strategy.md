# まちスタジオ Instagram運用戦略：ローカルメディア・ピボット（町の魅力発信）

## 1. 戦略の前提：なぜ「ローカルメディア化」なのか？
初期の「BtoBノウハウ特化（採用・集客の翻訳者）」から、**「町の魅力発信（ローカルメディア化）」**へ戦略を大きくピボットします。
この意図は、ランチェスター戦略に基づく「局地戦での圧倒的No.1（認知獲得）」と「事業の長期的資産化」にあります。

*   **目的1：圧倒的な「ポートフォリオ」としての機能**
    仙塩地区の魅力（店舗・企業・風景）をハイクオリティな映像や写真で発信し、それ自体を「まちスタジオの質の高い制作実績（技術証明）」とする。
*   **目的2：「影響力」をテコにした最強のBtoB営業（インバウンド獲得）**
    フォロワー1万人規模の「地域インフルエンサー」となることで、地元企業から「うちも取材・PRしてほしい」「自社アカウントの運用も任せたい」という相談を自然発生させる。（BtoC発信をBtoBリード獲得の装置にする）
*   **目的3：コアバリュー（地元への還元）の体現と熱量の維持**
    粕谷様自身の「地域貢献・塩竈への愛」に直結する発信であるため、運用自体に熱量がこもり、継続確率が飛躍的に高まる。

## 2. ターゲットとフェーズ展開

*   **ターゲット（閲覧者）**：仙塩地区を中心とした地元住民、カフェ・グルメ・おでかけ情報に関心がある層
*   **隠れターゲット（真の顧客）**：発信を見た地元の経営者、店舗オーナー、採用担当者

### ■ フェーズ1：認知拡大とフォロワー獲得（ローカルメディア期）
とにかく「仙塩地区で一番おしゃれで見やすい情報アカウント」を目指し、1万フォロワー（エリア内での圧倒的認知）を最速で取りに行く。
*   **発信内容**：仙塩地区のカフェ、飲食店、おでかけスポット、隠れた名店。
*   **KGI/KPI**：フォロワー数、保存数、リーチ数（地元ユーザーからのエンゲージメント）。

### ■ フェーズ2：影響力のマネタイズとBtoB展開（外部CMO期）
十分な影響力と「取材網（地元企業との接点）」ができた段階で、「まちスタジオ」としてのBtoBソリューション（映像制作、運用代行、採用ブランディング）を横展開する。
*   **発信内容**：「実は私たちが運営しています」という裏側発信、取材先企業の採用支援実績、店舗のPR支援実績など。
*   **KGI/KPI**：企業からの問い合わせ数、取材オファー数、リード獲得数。

## 3. ベンチマークすべき「地域発信アカウント」の条件

ノウハウ系アカウントではなく、以下のような**「地域メディアとして成功しており、かつクリエイティブの質が高いアカウント」**をベンチマークします。

1.  **写真・映像のクオリティで差別化しているアカウント**（ただの素人食レポではないプロの画作り）
2.  **マガジン風（まとめ・比較）で「保存」を量産しているアカウント**（例：「仙台の隠れ家カフェ5選」）
3.  **ショート動画（リール）でシズル感や店舗の空気感を伝えるのが上手いアカウント**

**【検索キーワードリスト（ベンチマーク発掘用）】**
*   `#仙台カフェ` / `#塩竈グルメ` / `#宮城観光` （タグ検索の「トップ」を見る）
*   `仙台 週末おでかけ` / `宮城 隠れ家`
*   `地域メディア` / `ローカルマガジン`（他県でもクリエイティブが優れたものを参考にする）
